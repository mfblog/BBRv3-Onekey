# ⚡ Debian / Ubuntu 一键安装 XanMod BBRv3 & 智能网络调优

一款专为 **Debian 12/13** 以及 **Ubuntu 20.04/22.04/24.04/26.04 (LTS)** 深度定制的 BBRv3 一键安装、升级与多场景网络队列调度（qdisc）双系统调优工具。

---

## 🌟 项目亮点

*   **双系统完美适配**：支持 Debian 12/13 和 Ubuntu LTS 常见版本（Debian 12 自动适配稳定 LTS 内核，Debian 13 及 Ubuntu 自动适配最新 Main 主线内核），稳定、高效、无后门。
*   **高精度战区判定**：集成 Cloudflare 边缘路由链路分析，100% 精准识别 VPS 出网物理归属，菜单自动高亮推荐最佳方案。
*   **4 重静默冗余绕过 WAF**：内置 UA 伪装、IPv4 降级与临时文件落地校验，并采用静默重定向技术。不仅能彻底解决阿里云香港、AWS JP等高危 IP 遭遇的 Cloudflare 403 证书阻断，还能保证终端界面无任何报错杂音，极致干净。
*   **智能版本比对与热应用**：若已是最新内核，自动跳过内核安装并立即热应用 `sysctl` 网络参数，修改配置免重启，即改即用。
*   **闭环一键体检 (5号选项)**：实时核对 `uname`、`sysctl` 及 `tc qdisc` 网卡调度层，完美闭环验证加速效果，并高亮输出内核与 BBRv3 运行状态。

## 📊 调优方案矩阵 (智能推荐)

脚本会根据您的 VPS 位置，在终端里**高亮加粗**并打上 `[⭐ 推荐]` 标签，您只需根据业务场景（建站/纯翻墙）二选一：

| 选项 | 适用场景 | 队列算法 (Qdisc) | 专属优化策略 |
| :--- | :--- | :--- | :--- |
| **1** | **[建站+翻墙] 亚太低延迟平衡型** (JP/SG/HK/KR 等) | `FQ_CODEL` | 适度亚太网络窗口缓冲，保护 MySQL 与建站内存不溢出 |
| **2** | **[建站+翻墙] 欧美远距离平衡型** (US/CA/EU/AU 等) | `FQ_CODEL` | 标准 TCP 缓存窗口，极低系统负载 |
| **3** | **[纯翻墙机] 亚太低延迟极限型** (JP/SG/HK/KR 等) | `CAKE` | 极限放大亚太 TCP 缓存窗口，全力抗晚高峰丢包与抖动 |
| **4** | **[纯翻墙机] 远距离吞吐极限型** (US/CA/EU/AU 等) | `FQ` | 针对长距离高延迟链路优化，压榨单线程大带宽吞吐上限 |
| **5** | **🔍 一键验证系统加速状态 (Status Check)** | - | 实时高亮输出内核、BBR 拥塞控制及网卡实际生效状态评估 |

## 🛡️ 系统安全保障 (逃生舱设计)

系统的包管理器在安装新内核时会**保留您原本的默认内核**。
万一遇到由于硬件/虚拟化不兼容导致开机死机（概率极低），您只需通过商家网页控制台（VNC/Console）重启服务器，在 GRUB 启动菜单中选择 `Advanced options` ➡️ 选中老内核（如 6.1.x 或 5.x-amd64），即可一键满血救活。
